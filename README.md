# AppGraph

AppGraph is an interactive canvas and dashboard application.

## Live Demo
Check out the live deployment here: [https://app-graph-eta.vercel.app/](https://app-graph-eta.vercel.app/)

## Overview
This application uses Mock Service Worker (MSW) to provide a rich, interactive demo environment entirely in the browser, without requiring a real backend server. All network requests are intercepted and handled by the local mock service worker, allowing you to explore the application's capabilities seamlessly.
